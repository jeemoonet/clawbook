# baidu-search：百度搜索

[← 返回技能列表](./index.md)

---

## 一句话用途

让 AI 助手**联网检索中文信息**，基于百度搜索能力（多通过百度千帆等开放接口），适合在飞书里问「最近有什么新闻」「查一下某某公司简介」等需要实时或事实类答案的问题。

---

## 适用场景

- **实时信息**：今日新闻、天气、股价、政策动向等。
- **事实查询**：公司介绍、产品说明、技术文档、百科类内容。
- **国内可用**：服务与数据在国内，无需境外网络，适合企业内网或合规要求高的环境。

---

## 安装

```bash
clawhub install baidu-search
# 或国内推荐
skillhub install baidu-search
```

---

## 配置

### 1. API 密钥

通常需要**百度千帆**或百度搜索开放平台的 API Key。获取方式：

- 登录 [百度智能云千帆](https://cloud.baidu.com/product/wenxinworkshop) 或百度搜索开放平台
- 创建应用，获取 API Key（或 Access Token）

### 2. 在 OpenClaw 中配置

在 `~/.openclaw/openclaw.json` 的 `skills.entries` 中为该技能配置密钥，例如：

```json5
{
  skills: {
    entries: {
      "baidu-search": {
        enabled: true,
        env: {
          BAIDU_API_KEY: "你的千帆或搜索 API Key",
        },
      },
    },
  },
}
```

也可用环境变量 `BAIDU_API_KEY`（变量名以技能 SKILL.md 为准）。修改后新会话生效。

## 怎么用

### 在对话中使用

用户直接在飞书（或其它已接入的渠道）里问即可；AI 会调用百度搜索能力检索后，结合结果组织回复，无需用户输入额外命令。

### 三个示例

**示例 1：行业新闻汇总**

- **用户**：「最近关于 AI 的行业新闻有哪些？」
- **Agent 实现**：识别为「实时/资讯类」→ 调用 baidu-search，传入查询词（如「AI 行业新闻 最近」）→ 技能用配置的百度/千帆 API 请求检索，获取网页或资讯结果列表（标题、摘要、链接）→ Agent 从中选取 3～5 条组织成「近期 AI 行业新闻：1. … 2. …」并附链接。
- **最终结果**：用户收到几条近期 AI 相关新闻的摘要与链接，点击可跳转原文。

**示例 2：公司信息查询**

- **用户**：「查一下某某公司的成立时间和主营业务。」
- **Agent 实现**：解析为事实查询 → 调用 baidu-search，查询「某某公司 成立时间 主营业务」→ API 返回相关摘要与链接 → Agent 从结果中抽取成立时间、主营业务等，组织成一段介绍并注明「来源：…」。
- **最终结果**：用户得到该公司成立时间与主营业务的简要介绍，无需自己搜索和翻页。

**示例 3：天气等即时信息**

- **用户**：「今天北京天气怎么样？」
- **Agent 实现**：识别为「天气、今日、北京」→ 调用 baidu-search 查询「北京 今天 天气」→ 从返回结果中抽取温度、晴雨、风力等 → Agent 回复「北京今天晴，X°C～Y°C，…」。
- **最终结果**：用户直接得到当日北京天气的简短描述。

### 能力说明

- 一般支持**网页、资讯**等检索类型（以接口为准）。
- 返回结果为摘要或链接，AI 会选取相关内容写入回复；若结果不理想，可让用户换一种问法或更具体的关键词。

---

## 注意事项

- **用量与限流**：百度接口有调用频次与配额限制，企业环境注意监控用量与成本。
- **内容合规**：检索结果来自公开网页，企业使用需符合内部内容与合规政策。
- **隐私**：避免在提问中包含敏感个人信息；必要时可关闭或限制该技能对部分群组开放。
