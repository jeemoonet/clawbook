# 企业应用手册（1️⃣ 企业应用）

本目录为以**企业应用**为总视角的 OpenClaw 飞书通道使用手册，全部为 `.md` 格式。

## 目录结构

| 文件 | 说明 |
|------|------|
| `index.md` | 企业应用手册首页与导航 |
| `01-企业应用概述与考虑角度.md` | 企业应用定义、落地需考虑的 10 个角度、手册结构 |
| `02-安装与运行.md` | 环境、安装、网关管理 |
| `03-飞书应用创建与发布.md` | 飞书企业自建应用、权限、事件订阅、发布与生命周期 |
| `04-通道配置与访问控制.md` | OpenClaw 配置飞书、私聊/群聊策略、允许列表（企业推荐） |
| `05-多团队与多Agent.md` | 按部门/群路由到不同 Agent、bindings |
| `06-安全与合规.md` | 凭证、访问控制、审计、合规与数据隐私 |
| `07-Skills与企业能力.md` | 国内可用的企业级 Skills 获取与安装 |
| `08-运维与故障排查.md` | 日志、监控、备份、常见问题与应急 |
| `10-配置参考.md` | 飞书通道配置项速查 |

说明：日常使用与命令已合并至 `02-安装与运行.md`。

## 与 docs/feishu 的关系

- **docs/feishu**：按「飞书通道」多角度拆分的通用手册（入门、飞书侧、配置、安全、企业、Skills 等）。
- **docs/企业应用**：以「企业应用」为**唯一目录**，将上述内容按企业落地顺序重写与合并，侧重访问控制、多团队、安全合规与运维。

二者可并存：需要按企业视角一条龙阅读时用本目录；需要按单主题（如仅飞书创建、仅安全）查阅时用 `docs/feishu`。
