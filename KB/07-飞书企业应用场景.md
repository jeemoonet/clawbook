# 飞书企业应用

## 飞书相关 Skills（示例）

以下为 ClawHub / 社区中与飞书深度集成的技能示例，安装前请在 SkillHub 或 ClawHub 搜索确认 slug 与版本。

| 技能名称（slug 示例）                             | 用途         | 说明                                                                                                           |
| ----------------------------------------- | ---------- | ------------------------------------------------------------------------------------------------------------ |
| **feishu-interactive-cards**              | 飞书交互卡片     | 在飞书中发送带按钮、表单、投票、待办等交互卡片，用户通过点击而非打字回复；适合确认、选择、问卷、任务列表。安装：`clawhub install feishu-interactive-cards`（需配合回调服务）。 |
| **feishu-calendar-tool**                  | 飞书日历       | 零配置读取 `openclaw.json` 中飞书凭证；创建、列表、删除日历日程，支持标题、时间、描述、地点、参与人、时区；带日历 ID 缓存。                                     |
| **feishu-calender** / **feishu-calendar** | 飞书日历自动化    | 创建日历/日程、查询空闲/忙碌、订阅日历变更；基于 Calendar v4 API，适合会议排期与冲突检测。                                                       |
| **feishu-api-bitable**                    | 飞书多维表格 API | 多维表格的增删改查：表、记录、字段的创建与维护；需配置 `FEISHU_APP_ID`、`FEISHU_APP_SECRET`。                                             |
| **feishu-bitable-creator**                | 飞书多维表格创建   | 从零创建可用的多维表格，自动清理占位行列、批量创建字段与记录；需飞书渠道配置及 `bitable:app`、`drive` 等权限。                                           |

以上技能与 OpenClaw 飞书通道共用同一套飞书应用凭证时，通常从 `~/.openclaw/openclaw.json` 的 `channels.feishu` 读取；若需单独配置，见各技能的 SKILL.md。企业环境部署交互卡片等涉及回调的技能时，需注意回调安全与权限（参见各技能安全说明）。

**每项技能的使用说明（用途、安装、配置、怎么用）** 见 [常用 Skills 使用说明](./skills/index.md)。

## 飞书官方能力与典型应用场景

飞书开放平台提供的能力可与上述 Skills 或自建逻辑结合，形成完整企业场景。

| 场景                | 飞书官方能力                  | 说明                                                      |
| ----------------- | ----------------------- | ------------------------------------------------------- |
| **机器人**           | 企业自建应用机器人、群 Webhook 机器人 | 私聊与群聊 @ 机器人、消息收发；OpenClaw 飞书通道即基于企业自建应用 + 长连接。          |
| **云文档**           | 云文档 API（新版/旧版）          | 创建、编辑、权限管理、元数据、插入/更新内容；适合自动化报告、模板生成、文档协作。               |
| **多维表格（Bitable）** | Bitable API             | 表/记录/字段的 CRUD；适合工单、项目看板、数据汇总与报表。                        |
| **日历与日程**         | 日历 API（Calendar v4）     | 日程创建与管理、空闲/忙碌查询、会议室集成；适合会议预约、排期、与 Pad 展示联动。             |
| **知识库 / 知识空间**    | 知识库 API                 | 文档挂载到知识空间、权限继承；适合内部知识沉淀与检索。                             |
| **审批**            | 审批 API                  | 创建/查询审批实例、多种表单控件（文本、数字、联系人、部门、时间、明细、附件等）；适合请假、报销、流程类场景。 |
| **通讯录**           | 通讯录 API                 | 用户列表、部门结构、批量获取用户；适合组织架构同步、@ 提及解析、权限与可见范围。               |
| **消息与群组**         | 消息 API、群管理              | 发送文本/富文本/卡片、群成员与群信息；与机器人能力配合实现通知、互动与审批提醒。               |

在 OpenClaw 中：**通道**负责在飞书里收发消息（本手册已覆盖）；**Skills** 或自定义工具则可在此基础上调用上述 API，实现「在飞书中对话即可查日历、写文档、填多维表格、发起审批」等场景。应用需在 [飞书开放平台](https://open.feishu.cn/app) 申请对应权限并发布。

## 安全注意

- 第三方技能视为**不可信代码**：安装前阅读 `SKILL.md` 与脚本，再启用。
- 技能会获得你配置的 `env`/`apiKey`，勿将敏感信息写入日志或提示词。
